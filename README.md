# Лабораторные работы
Используется в рамках занятий в ГБОУ «Бауманская инженерная школа №1580». Данный репозиторий содержит лабораторные работы, выполненные в рамках учебного процесса.

## Назначение
Репозиторий используется для:
- хранения лабораторных работ;
- хранения учебных материалов;
- передаче материалов преподователю;

Проект служит для систематизации выполненных лабораторных работ, хранения сопутствующих учебных материалов и заметок, оперативной передачи готовых решений на проверку преподавателю.


### Данный репозиторий является частным учебным архивом. Он не предназначен для публичного использования, копирования или распространения сторонними лицами.
